# JLU_lessons
Select Lessons Program for JLU

JLU吉林大学选课程序 Wrote by Ruinous

Update：2019/12/19 17:00

## 使用方法

请使用Python3运行，并且安装requests库

Usage: python qk.py

请提前把需要选的课程加入到快捷选课列表再运行本脚本

抢课时间以本地时间为准，请自行调至准确北京时间
